## Running Locally
Clone the Repo:
```bash
git clone https://github.com/tr3m/bloodlevel-calculator
```

Install the dependencies:
```bash
cd bloodlevel-calculator
npm install
```

Start the development server
```bash
npm start
```

## Building
For a production build run:
```bash
npm run build
```
